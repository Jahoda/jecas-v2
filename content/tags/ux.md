<p>UX (<i lang="en">User Experience</i>) se zabývá tím, jak uživatel vnímá a používá webovou stránku. Cílem je vytvořit intuitivní, rychlé a příjemné rozhraní.</p>

<p>Dobrý UX zahrnuje srozumitelnou navigaci, čitelnou <a href="/typografie">typografii</a>, <a href="/responsive">responsivní design</a>, rychlé <a href="/zrychlovani">načítání</a> a přístupnost pro všechny uživatele.</p>
