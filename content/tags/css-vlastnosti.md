<p>CSS vlastnosti (properties) definují vizuální vzhled HTML elementů – od barev a rozměrů přes <a href="/layout">rozložení stránky</a> až po <a href="/webove-animace">animace</a> a přechody.</p>

<p>Každá vlastnost přijímá určité typy hodnot a může být zděděna z nadřazeného prvku. Moderní <a href="/css">CSS</a> přináší stále nové vlastnosti pro flexibilnější a výkonnější stylování.</p>
