<p>Animace na webu lze vytvářet pomocí <a href="/css">CSS</a> (vlastnosti <a href="/animation">animation</a> a <a href="/transition">transition</a>) nebo <a href="/js">JavaScriptu</a> (Web Animations API, requestAnimationFrame).</p>

<p>CSS animace jsou výkonnější a preferované pro jednoduché efekty, JavaScript se hodí pro složitější interaktivní animace řízené uživatelem.</p>
