<p>AJAX (Asynchronous JavaScript and XML) je technika pro asynchronní komunikaci se serverem bez nutnosti znovunačtení celé stránky. Umožňuje dynamicky načítat a odesílat data na pozadí.</p>

<p>Moderní <a href="/js">JavaScript</a> používá pro AJAX volání metodu <code>fetch()</code>, která nahradila starší objekt <code>XMLHttpRequest</code>. Data se dnes nejčastěji přenášejí ve formátu JSON místo původního XML.</p>
