<p>Git je distribuovaný verzovací systém pro správu zdrojového kódu. Umožňuje sledovat historii změn, spolupracovat v týmu a snadno se vracet k předchozím verzím.</p>

<p>Nejpoužívanější platformy pro hostování Git repozitářů jsou GitHub, GitLab a Bitbucket.</p>
