<p>Změna vzhledu webu přepínáním <a href="/css">CSS</a> tříd <a href="/js">JavaScriptem</a>. Typicky dark mode / light mode přepínač, barevná témata nebo zvětšení písma pro lepší čitelnost.</p>

<p>Moderní přístup využívá <code>prefers-color-scheme</code> media query pro automatickou detekci systémového nastavení.</p>
