<p>CSS pravidla (at-rules) jsou speciální příkazy začínající znakem <code>@</code>, které řídí chování stylů. Patří mezi ně například <code>@media</code> pro <a href="/responsive">responsivní design</a>, <code>@keyframes</code> pro <a href="/webove-animace">animace</a> nebo <code>@import</code> pro načítání externích stylopisů.</p>

<p>Pravidla umožňují podmíněné aplikování stylů, definici fontů pomocí <code>@font-face</code> a další pokročilé techniky v <a href="/css">CSS</a>.</p>
