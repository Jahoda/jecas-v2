<p>CSS funkce jsou speciální výrazy, které se používají jako hodnoty <a href="/css-vlastnosti">CSS vlastností</a>. Umožňují dynamické výpočty, transformace a pokročilé efekty přímo v <a href="/css">CSS</a>.</p>

<p>Mezi nejznámější patří <code>calc()</code> pro matematické výpočty, <code>var()</code> pro práci s CSS proměnnými, <code>rgb()</code> a <code>hsl()</code> pro definici barev nebo <code>url()</code> pro načítání externích zdrojů.</p>
