<p>Layout (rozložení stránky) je způsob, jakým jsou na stránce uspořádány jednotlivé bloky obsahu. Moderní <a href="/css">CSS</a> nabízí pro tvorbu layoutu zejména Flexbox a CSS Grid.</p>

<p>Správně zvolený layout zajišťuje přehlednost stránky na všech zařízeních a je základem <a href="/responsive">responsivního designu</a>.</p>
