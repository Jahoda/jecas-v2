<p>SQL (Structured Query Language) je standardní jazyk pro správu a dotazování relačních databází. Umožňuje vytvářet, číst, upravovat a mazat data pomocí příkazů jako <code>SELECT</code>, <code>INSERT</code>, <code>UPDATE</code> a <code>DELETE</code>.</p>

<p>Na webu se SQL nejčastěji používá prostřednictvím databází MySQL nebo PostgreSQL, v <a href="/php">PHP</a> například přes rozhraní <a href="/php-pdo">PDO</a>.</p>
