<p>Fungování webové stránky bez připojení k internetu (offline). Díky technologiím jako Service Worker a Cache API lze zajistit, že web zůstane dostupný i bez síťového připojení.</p>

<p>Offline podpora je klíčovou součástí progresivních webových aplikací (PWA) a zlepšuje <a href="/ux">uživatelský zážitek</a> zejména na mobilních zařízeních s nestabilním připojením.</p>
