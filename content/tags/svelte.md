<p>Svelte je moderní JavaScript framework pro tvorbu uživatelských rozhraní. Na rozdíl od React nebo Vue kompiluje komponenty do optimalizovaného JavaScriptu během buildu.</p>

<p>SvelteKit je full-stack framework postavený na Svelte, podobně jako Next.js pro React.</p>
