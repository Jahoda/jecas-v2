<p>Je to nástroj, který umožňuje vytvářet, testovat a dokumentovat HTTP požadavky a odpovědi. Může být použit pro vývoj a testování různých typů webových služeb – jako jsou <b>REST</b>, <b>SOAP</b>, <b>GraphQL</b> atd.</p>

<p><a href="https://www.postman.com">Postman</a> má řadu užitečných funkcí, jako je schopnost ukládat požadavky a odpovědi, automatisovat testování a integrovat se s dalšími nástroji pro vývoj aplikací.</p>

<p>Je k dispozici jako rozšíření pro webové prohlížeče i jako samostatná aplikace pro <b>Windows</b>, <b>macOS</b> a <b>Linux</b>.</p>

<p><img src="/files/postman/postman.png" alt="Postman" class="border"></p>


<h2 id="pro">Pro koho se hodí</h2>

<p>Hodí se pro vývojáře, kteří vyvíjejí API, a v Postmanovi si snadno ověří správný výsledek.</p>

<p>Užitečný je i pro <b>testery</b>, pro otestování, že API funguje, jak má.</p>

<p>Užitečný je i při implementaci a integraci cizích API, protože si v něm člověk může snadno odladit potřebná API volání.</p>


<h2 id="alternativy">Alternativy</h2>

<p>Existuje mnoho alternativ nabízející podobné funkce, například:</p>

<ol>
  <li>
    <p><a href="https://curl.se"><b>cURL</b></a> – příkazový nástroj, který umožňuje odesílat HTTP požadavky z příkazového řádku. Je to příkazový nástroj, takže ho mohou používat především vývojáři a operátoři systémů.</p>
  </li>
  
  <li>
    <p><a href="https://www.soapui.org"><b>SoapUI</b></a> – nástroj pro testování webových služeb, který je k dispozici pro Windows, MacOS a Linux. Poskytuje širokou škálu nástrojů pro testování a automatizaci, včetně možnosti vytvářet testovací scénáře a sledovat výsledky.</p>
  </li>
  
  <li>
    <p><a href="https://insomnia.rest"><b>Insomnia</b></a> – je další nástroj pro testování a dokumentaci API, který je k dispozici jako samostatná aplikace pro operační systémy Windows, MacOS a Linux. Má jednoduché a intuitivní rozhraní a poskytuje širokou škálu nástrojů pro testování a automatizaci.</p>
  </li>
  
  <li>
    <p><a href="https://paw.cloud"><b>Paw</b></a> – nástroj pro testování a automatizaci API, který je k dispozici pro MacOS. Má intuitivní rozhraní a poskytuje širokou škálu nástrojů pro testování a automatizaci, včetně možnosti vytvářet testovací scénáře a sledovat výsledky.</p>
  </li>
  
  <li>
    <p><a href="https://www.telerik.com/fiddler"><b>Fiddler</b></a> – nástroj pro monitorování, ladění a testování HTTP požadavků, který je k dispozici pro Windows. Může být použit pro testování různých typů webových služeb a poskytuje širokou škálu nástrojů pro ladění a monitorování HTTP požadavků.</p>
  </li>
</ol>
